Primary request: Create a polished fictional game sheet titled "Backstage Leak Hunt" derived from the Service Blueprint Distiller concept. Players inspect a service blueprint and repair five logic errors where internal work leaks into customer view or the map confuses touchpoints, causes, fixes, and outcomes.
Scene/backdrop: flat printable facilitation sheet on clean warm paper, straight-on, no people, no hands, no real brand. It should feel like a workshop puzzle, not a software dashboard and not a board game photo.
Subject: center shows a simplified fictional Mossline Library Pickup blueprint strip with lanes: "Customer", "Frontstage", "Visibility line", "Backstage", "Support". Five circled error markers appear on the strip with small repair tokens beside it.
Game mechanism: include a concise rule rail with five steps: "1 Spot", "2 Name", "3 Move", "4 Assign", "5 Retest". Show five safe fictional error cards: "internal note visible", "missing support link", "emotion in wrong stage", "fix has no owner", "outcome used as cause". Show a small solved-state preview with green checks, and a timer badge "8 min".
Visual format: landscape single-sheet activity with one long blueprint strip, side repair-token column, and bottom scoring row. No 2x2 or 2x3 grid. No product mockup, no physical controller, no tabletop props, no UI dashboard.
Style/medium: crisp editorial UX workshop material, playful but professional, clean iconography, clearly separated lanes, high legibility, subtle paper texture.
Color palette: warm white, graphite, teal, coral error circles, green pass checks, muted yellow repair tokens, small blue owner dots. Avoid purple-dominant palettes.
Text requirements: render only short labels listed in this prompt; keep text mostly readable and avoid dense paragraphs.
Safety and rights constraints: completely fictional service; no real brands, logos, institutions, customer names, phone numbers, addresses, QR codes, barcodes, public figures, politics, medical/legal/financial service claims, harmful instructions, adult content, gore, weapons, copyrighted characters, celebrity likeness, living-artist style imitation, watermark, copied template, copied source image, or external prompt text.
