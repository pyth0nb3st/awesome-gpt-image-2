Full Prompt

Create one polished public-gallery image titled "Derived Play: Cushion Stack Trial".

It is derived from Prompt Crate Test but must become a different playable object: a cooperative tabletop stacking-and-inspection challenge using modular cushion blocks, not the same open crate, not a grid, not a contact sheet, not cards-only, not a dashboard, not a transit concourse, not a bakery deck, not a snowglobe rack, not a water basin, not a wardrobe carousel, not a fossil drawer, not a toy latch.

Output format: one square 1:1 overhead product photograph of a circular cork test pad with a low central pedestal, a ring of soft foam cushion blocks, tiny tilt tokens, paper inspection tickets, source seal tabs, and a clear arrival loupe. No people, no hands, no logos, no real shipping labels, no real brands.

Play rules shown through objects: players place the blue ribbon prism on the pedestal, draft Cushion Blocks around it, rotate Context Corners, slide one Edit Hatch tile under the pedestal, spend Void Fill plugs to block extra shapes, attach one Caption Label ticket, pass the Aspect Hoop over the stack, wrap one Style Sleeve, then score Arrival Proof through the loupe only if the same three-finned blue ribbon prism with the amber dot remains visible and centered after the tilt token is moved.

Short visible labels: Cushion Draft, Corner Rotate, Edit Tile, Void Plug, Caption Ticket, Aspect Hoop, Style Sleeve, Tilt Token, Arrival Proof. Visual language: premium tactile tabletop prototype, cork circle, recycled grey foam, porcelain blue prism, amber dot, teal paper seals, brass loupe, crisp readable labels, playful but sophisticated, fictional, original, unbranded, harmless, non-political, non-adult, non-graphic, rights-safe.
