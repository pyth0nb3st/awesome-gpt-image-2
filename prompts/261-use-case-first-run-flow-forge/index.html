Full Prompt

Use case: ui-mockup
Asset type: GPT Image 2 use-case gallery image, wide 16:9 UX prototype review artifact
Primary request: Create a polished fictional mobile onboarding prototype review image titled "First-Run Flow Forge". Show four tall phone-screen mockups connected left to right by a thin activation ribbon for a fictional urban-garden planning app named "Pocket Orchard". The screens should be original and not resemble any real app: 1) welcome screen with clear "Skip" and "Start" choices, 2) one-question setup about balcony sunlight with simple segmented choices, 3) empty garden state with one strong CTA "Add first plant", 4) contextual reminder pre-prompt with "Not now" and "Enable later" options, followed by a small success plant card.
Composition: professional product-design review board, but not a 2x2 or 2x3 grid; one continuous horizontal flow with phone frames, light connector arrows, small numbered touch targets, and a right-side QA rail.
QA rail text: "value first", "one question", "empty state CTA", "pre-prompt", "skip path", "mobile-native". Include small pass/fix chips, activation checkpoints, and a compact footer stamp "fictional app / no real brand / no source image".
Visual style: crisp modern UX design artifact, readable short UI copy, soft off-white canvas, graphite annotations, muted garden greens, coral warnings, clean type, subtle shadows, high polish.
Avoid: no real brands, no real platform logos, no app-store badges, no real people or faces, no public figures, no politics, no medical/legal/financial advice, no copyrighted characters, no copied UI, no dense tiny text, no lorem ipsum, no dashboard, no menu, no map, no physical apparatus, no 2x3 research board.
