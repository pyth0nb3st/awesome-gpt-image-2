Full Prompt

Create a polished GPT-image-2 use-case research board as one single 3:2 landscape image, like a premium design lab documentation wall. Overall title: "GPT-IMAGE-2 USE CASE LAB: LIVE CONTEXT LOOPS". Six clearly separated panels in a clean 3x2 grid, each panel with a short readable English title and a concrete demo. These must be NEW use cases, avoiding prior recorded gallery themes: multilingual posters, product SKU factories, UI fiction simulators, storyboard continuity, educational remixes, interior matrices, prompt escape rooms, brand multiverses, time machines, recipe comics, worldbuilding atlases, catalog transformers, data posters, onboarding state machines, localized packaging, restoration what-ifs, game asset sheets, prompt archaeology, QA spot-the-diff, microbrand kits, learning cards, wayfinding remix, tabletop quest props, process twins, field notebooks, policy theater, kitchen mise maps, archive color labs, failure mode atlases, state lock, evidence garden, scenario switch, prep race, palette detective, failure ladder, field note builder, constraint posterizer, recipe reality check, policy comic translator, archive label restorer, failure mode catalog, observation bingo, constraint auction, recipe forensics, policy storyboard race, archive detective, glitch taxonomy draft.

Panel 1 title: "Calendar Scene Forge" showing a week of plain calendar events transformed into consistent location thumbnails, outfit hints, weather badges, and time-of-day lighting cues.
Panel 2 title: "Meeting Memory Board" showing rough meeting notes converted into a visual decision map with faces as neutral silhouettes, action cards, blockers, and ownership chips.
Panel 3 title: "Dataset Postcard Sampler" showing rows from a small fictional spreadsheet turned into six mini postcards, each preserving numeric facts as readable labels and icons.
Panel 4 title: "Repair Triage Photos" showing user-submitted appliance photos becoming a safe diagnostic photo checklist with numbered areas, severity tags, and "call expert" flags.
Panel 5 title: "Language Mood Matrix" showing the same short message rendered as visual tone cards across formal, friendly, urgent, playful, and calm moods in multiple scripts, no real brands.
Panel 6 title: "Spec-to-Shelf Prototype" showing a plain product requirements spec converted into shelf-ready rough mockups: box, label, insert, warning card, and comparison tag.

Style: premium editorial documentation, realistic printed boards, crisp cards, mini screenshots, tiny charts, material samples, sticky notes, calipers, pencil annotations, warm studio light. Rich varied accent colors but not dominated by one hue. Typography should be readable only for overall title, six panel titles, and short labels like "TODO", "RISK", "OWNER", "FACT", "SAFE", "DRAFT". No real logos, no watermarks, no copyrighted characters, no celebrity likenesses.
