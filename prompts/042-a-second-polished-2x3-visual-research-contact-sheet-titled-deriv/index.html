Full Prompt

Create a second polished 2x3 visual research contact sheet titled "Derived Play Patterns for GPT Image 2.0 / 派生玩法". Aspect ratio 16:9, visually rich, each panel a different playful product/game-like concept built from image generation capabilities. Include clean readable short labels: 1) "REMIX DICE / 变体骰子" showing dice that randomize style, season, camera, and audience for one product image; 2) "VISUAL QUEST / 视觉任务" showing a small quest board where each generated image unlocks the next prompt clue; 3) "BRAND NPC / 品牌角色" showing one mascot character adapted into package, sticker, app avatar, and comic; 4) "MEMORY PALACE / 记忆宫殿" showing a study room where facts become surreal visual landmarks; 5) "A/B ARENA / 视觉竞技场" showing four ad thumbnails competing with score badges like clarity, novelty, click intent; 6) "WORLD KIT / 世界观素材包" showing map, props, character sheet, item cards, and poster in one coherent fictional world. Use original fictional content only, no real brands, no logos, no watermark. Emphasize structured layouts, consistent characters/assets, legible bilingual headings, and playful creative direction.
