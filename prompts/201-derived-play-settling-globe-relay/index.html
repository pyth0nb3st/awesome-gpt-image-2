Full Prompt

Create one polished public-gallery image titled "Derived Play: Settling Globe Relay".

This is a derived playable mode from Prompt Settling Chamber, but it must look like a different rules object: a cooperative shake-and-observe tabletop relay using five small transparent snowglobes in a curved rack. It is not the same single cutaway globe, not a grid, not a contact sheet, not cards, not a rectangular board game, not a cabinet, not a water basin, not a wardrobe, not a fossil drawer, not a toy latch, not a greenhouse, not a weather station, not a planetarium, not a music box, not a flipbook, not a loom, not a proof press, not an optical bench, not a magnetic field device.

Output format: one wide 16:9 low-oblique product photograph of a crescent-shaped tabletop rack holding five palm-size clear globes. Each globe contains a tiny version of the same invented ceramic moon-harbor with three teal archways and one amber beacon, plus different harmless flakes and transparent rings. Around the rack are a sand timer, small shake tokens, caption clips, mask sleeves, style beads, and a final proof loupe. No people, no hands, no real places, no logos.

Playable rules shown through objects: players pass a globe along the crescent rack, spend one Shake Token, add or remove one Edit Flake, slide one Mask Sleeve, choose one Style Bead, clip one Caption Tag, align one Crop Ring, then wait for the Settling Timer. A point scores only if the proof loupe still sees the same three teal arches and amber beacon after the flakes settle.

Readable short labels on components: Shake Token, Edit Flake, Mask Sleeve, Style Bead, Caption Tag, Crop Ring, Settling Timer, Proof Loupe, Relay Score. Visual language: premium tactile tabletop prototype, graphite crescent rack, clear glass globes, teal ceramic harbors, amber beacon glow, pearl flakes, transparent acrylic rings, precise tiny labels, crisp studio reflections, playful but sophisticated. Entire concept is fictional, original, unbranded, family-safe, non-graphic, and rights-safe.
