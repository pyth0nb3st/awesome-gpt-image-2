Use case: scientific-educational / productivity-visual
Asset type: tall isometric learning-map artifact, 4:5 portrait.
Title text on the image: "Memory Palace Mapper".
Primary request: Show a fictional study workflow where GPT Image 2 turns a dense paragraph of notes into a navigable memory palace. The image should look like one coherent cutaway map of an imaginary study loft, not a 2x3 board and not a product photo. Include six numbered loci along a clear walking route: Door, Window, Desk, Shelf, Lamp, Balcony. Each locus has a small invented cue scene and a tiny source-note snippet card attached. Include a narrow right-side review rail with five checklist chips: ORDER, BINDING, DISTINCT CUE, SOURCE LINK, RECALL TEST. Include a bottom strip showing the workflow: NOTES -> ANCHORS -> ROUTE -> CUES -> TEST.

Visual language: clean editorial isometric illustration mixed with crisp UI annotation; warm white paper background, teal route arrows, coral cue markers, charcoal labels, muted green review rail. Text should be short, legible, and fictional. Make the artifact feel useful for students and teachers without showing any real people.

Safety and originality constraints: no real school, university, brand, logo, person, face, address, copyrighted character, celebrity, political content, medical or therapeutic efficacy claim, dangerous instruction, source-image reuse, copied prompt, copied sketchnote, copied gallery image, or living-artist style imitation. Avoid ornate fantasy palace cliches; emphasize spatial mnemonic structure and reviewable output.
