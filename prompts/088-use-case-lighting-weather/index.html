Use case: lighting-weather
Asset type: derived play concept image, single finished image
Primary request: Create a 16:9 interactive concept called "LIGHTING DJ", derived from the same-room four-hours image. Show a tactile console that lets players remix one room's lighting with sliders for sun angle, lamp warmth, window weather, and shadow softness, while preview thumbnails update around it.
Scene/backdrop: design studio desk with a physical control board and four preview screens of the same reading nook.
Subject: knobs and sliders labeled for lighting variables, thumbnails showing consistent room with different light recipes.
Style/medium: cinematic product photography mixed with clear UI screens.
Composition/framing: console centered, preview screens behind, hands absent.
Lighting/mood: moody studio with colorful but restrained light accents.
Color palette: charcoal console, white labels, amber lamp glow, cool blue weather preview, green plant accents.
Text (verbatim): "LIGHTING DJ", "Sun angle", "Lamp warmth", "Rain", "Shadow soft", "Preview".
Constraints: no real brands, no logos, no watermark, keep the same room layout in all thumbnails, labels should be readable.
