Full Prompt

Create a wide 16:9 original GPT Image 2 use-case concept scene titled "Use Case Patchbay: Prompt Signal Debugging".

Purpose: visualize a fresh set of GPT Image 2 use cases as a modular signal-debugging studio for image prompts. This must be one continuous wall-sized patchbay and oscilloscope workstation, not a 2x3 board, not a card spread, not a craft loom, not a planetarium, not a projector, not a printshop, not a weather station, not a film stage, not a tabletop game.

Scene format: a fictional rights-safe image lab where abstract prompt fragments are treated like colored signals. Show a tall modular patch panel with braided cables, small oscilloscopes, meters, toggle switches, test thumbnails, warning lamps, and a clean workbench below. All brands, logos, people, organizations, and labels are fictional or absent.

Six use-case modules should be integrated into the single patchbay as distinct controls, each with a short readable fictional label:
1. "ambiguity scope" - split one vague prompt into clearly different visual interpretations before choosing one.
2. "constraint mixer" - balance material, mood, framing, and detail constraints without overloading the image.
3. "conflict fuse" - reveal prompt instructions that fight each other, shown as a safe tripped fuse and a clean rewritten signal.
4. "series checksum" - keep a fictional object family visually consistent across multiple outputs using repeatable feature marks.
5. "iteration diff" - compare two generated attempts with tiny lit difference traces, without making a spot-the-diff board.
6. "thumbnail stress" - test whether a concept still reads at icon, sticker, poster, and banner sizes.

Visual language: polished product-design 3D illustration with editorial clarity; graphite panels, off-white labels, colored cables in teal, red, yellow, and green; glowing waveform lines, crisp tiny thumbnails, tactile knobs and switches; high detail but organized.

Composition: dramatic diagonal perspective along the patchbay wall, with modules connected by visible cable routes and small output screens. The image should feel like a working instrument for debugging prompts, not a poster of examples.

Safety and originality constraints: original concept only; no external prompt or image source; all entities, marks, characters, places, labels, and institutions must be fictional or rights-safe; avoid disallowed sensitive, unsafe, adult, graphic, real-world identity, brand-restricted, character-restricted, living-artist-style, and watermark material.

Do not include: underwater sonar, bathymetry, portholes, reference locks, mask rings, crop windows, compression tests, 2x3 research board, tabletop card spread, greenhouse, loom, weather room, letterpress, planetarium, lantern slide stack, film set, airport terminal, laundromat, pinball table, or star map.
