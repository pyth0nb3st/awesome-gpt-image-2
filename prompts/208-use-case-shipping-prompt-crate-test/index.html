Full Prompt

Create one polished public-gallery concept image titled "Use Case Shipping: Prompt Crate Test".

Output format: one 3:2 low-oblique product photograph of a fictional miniature shipping stress-test crate opened on a matte studio table. It is not a grid, not a contact sheet, not a dashboard, not cards, not a board game, not a tray, not a cabinet, not a transit concourse, not a bakery deck, not a snowglobe, not a water basin, not a wardrobe, not a fossil drawer, not a toy latch, not a loom, not a proof press, not a patchbay, not an elevator shaft.

Scene: an unbranded open crate made from pale basswood and soft recycled paper foam. The invented subject is a tiny ceramic "blue ribbon prism" with three rounded fins and one amber dot, nested inside a custom foam cutout. Around it are harmless miniature props: corner blocks, a reference packing slip, one hinged edit hatch, removable void-fill plugs, caption label strips, aspect-ratio crate gauges, style tissue wraps, tamper-evident source seals, and an arrival inspection window. No people, no hands, no real shipping labels, no logos.

Mechanism: GPT Image 2 prompt control becomes a packaging robustness test. Short readable labels on physical pieces: Subject Nest, Context Corners, Reference Slip, Edit Hatch, Void Fill, Caption Label, Aspect Gauge, Style Wrap, Source Seal, Arrival Proof. The same blue ribbon prism must stay recognizable after every packaging stress: corner blocks test context changes, the slip tests reference identity, the hatch tests local edits, void fill blocks unwanted elements, caption labels test text, aspect gauges test framing, tissue wraps test style constraints, source seals mark provenance, and the inspection window repeats the object exactly.

Visual language: premium tactile studio photography, pale wood crate, recycled grey paper foam, porcelain blue prism, amber dot, small brass gauges, teal inspection stickers, crisp readable labels, soft warehouse-window light, sophisticated playful artifact. Entire concept is fictional, original, unbranded, harmless, non-graphic, and rights-safe.
