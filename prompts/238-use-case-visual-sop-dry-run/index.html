Full Prompt

Create a polished portrait 4:5 concept image for a GPT Image 2 use-case gallery.

Title visible in the artwork: "VISUAL SOP DRY RUN".

Scene: a clean printable one-page operating guide for a fictional community event called "LUMEN LANE BOOK SWAP" shown as a realistic flat document preview on a neutral editor canvas, not a physical desk photo. The document has six illustrated numbered steps in a flowing vertical sequence, each step with one small icon, one short role label, one time chip, and one concise action line. Include only invented, non-branded labels such as "SETUP", "SORT", "SIGN", "WELCOME", "CHECK", "CLOSE" plus tiny bilingual support labels like "READY / 准备" and "PASS / 通过" where legible.

Mechanism to make visible: GPT Image 2 is being used to dry-run whether a generated instruction sheet has readable text, correct sequence, clear roles, and reviewable visual hierarchy. Add subtle human QA markup in a different ink color: checkmarks on steps 1, 2, 4, 6; small repair notes on step 3 reading "TEXT TOO SMALL" and on step 5 reading "ROLE DRIFT"; a bottom audit strip with five chips: "OCR", "ORDER", "ROLE", "ACCESS", "PASS". Make the artwork look like a production review artifact, not an ad poster and not a dashboard.

Visual style: contemporary editorial document design, off-white paper, charcoal text, muted teal and coral accents, hand-drawn but precise icons, slight human imperfection in annotation strokes, crisp readable typography, generous margins, no logos, no real brands, no real people, no copyrighted characters, no politics, no harmful instructions, no gore, no explicit content. Avoid a 2x2 or 2x3 grid board, avoid tabletop game pieces, avoid lab instruments, avoid macro product photography.
