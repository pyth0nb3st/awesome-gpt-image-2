Full Prompt

Create one polished public-gallery concept image titled exactly "Use Case: Style Drift Flight Recorder".

Output format: one ultra-wide 21:9 creative-operations flight-recorder strip, a continuous trace artifact for GPT Image 2 visual consistency testing. It must not be a physical tabletop object, not a device, not a tray, not a lab bench, not a contact sheet, not a comic, not a map, not a worksheet, not a generic SaaS dashboard. It should feel like a long editorial control-room wall display and consistency log for a fictional visual system.

Scene: a fictional no-brand public visual system called "Mosslight Festival". Show a row of small generated asset thumbnails embedded inside one continuous recorder timeline: poster crop, way sign crop, social tile crop, badge crop, email hero crop, and venue banner crop. The assets share the same invented visual contract: moss green, warm amber, ivory paper, graphite linework, one simple lantern-leaf icon, centered horizon, 35mm documentary framing, soft side light. No real logos, no real event, no real city.

GPT Image play mechanism shown visually: a drift waveform runs across the whole strip and compares each generated asset to the style contract. Add compact overlay labels only: "Use Case: Style Drift Flight Recorder", "Mosslight Festival", "Style Contract", "Palette", "Lens", "Anchor", "Text", "Count", "Position", "Rights Log", "Drift 04", "PASS". Show tiny QA chips: green pass chips for Palette, Lens, Anchor; one coral warning chip at Drift 04; one small rights-log tick. Keep all text short, readable, and exactly as listed; do not add random placeholder text.

Composition: a single connected horizontal artifact with a left style-contract panel, central drift trace waveform, asset thumbnails integrated into the trace, and a right pass stamp. Use precise alignment, hairline guides, small feature tags, mini color swatches, visual comparison brackets, and subtle scanline/ruler marks. The scene should communicate that GPT Image 2 can generate a campaign system and then audit consistency, attribution, and failure modes.

Visual style: high-end editorial information design mixed with soft cinematic thumbnails; matte off-white background, graphite typography, moss green linework, amber highlights, ivory cards, restrained coral warning, no purple-dominant gradient, no decorative orbs. Full-bleed, crisp, gallery-worthy.

Safety and rights: fictional visual system only, no real brands, no trademarks, no public figures, no politics, no elections, no dangerous instructions, no adult or explicit content, no gore or violence, no copyrighted characters, no living-artist style imitation, no source image reuse, no watermark.
