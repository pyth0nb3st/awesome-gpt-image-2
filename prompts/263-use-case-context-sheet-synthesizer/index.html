Full Prompt

Create a single polished GPT Image 2 use-case image titled visually as a fictional archival field-record sheet, not a product photo and not a grid board.

Use case name: Context Sheet Synthesizer.

Primary scene: a tall portrait cultural-heritage documentation dossier for a completely fictional excavation training site code "FEN-42". The image should look like one coherent printed field-record page laid flat in clean daylight, with no hands, no people, no real institution logos, no real archaeological site, no real artifact, and no culturally specific motif.

Core visual structure:
- Top strip: a short source note block labeled "FIELD NOTE" with generic invented text fragments, then an arrow to "STRUCTURED RECORD".
- Main left area: clean technical plan view of a square trench with abstract soil patches, context numbers C101, C102, C103, north arrow, scale bar, and a simple key for deposit, cut edge, stone, charcoal fleck. Keep shapes generic and invented.
- Main right area: vertical section drawing with layered soil bands labeled Layer A, Layer B, Layer C, with crisp boundaries and measurement ticks.
- Lower band: mini relationship matrix using simple boxes C101 above C102, C102 fills C103, plus sample/photo cross-reference chips S-02, P-07, D-03.
- Side review rail: five compact checks with readable labels: PLAN, SECTION, CONTEXT, SAMPLE, RELATION.
- Footer stamp: "fictional training record - no source image".

Style: publication-quality technical illustration mixed with modern archive layout, off-white paper, fine black ink, muted earth accent colors, measured lines, clean typography, high legibility, restrained but rich design. Avoid 2x3 contact sheet composition, avoid dashboard look, avoid macro product shot, avoid physical apparatus, avoid real brands, avoid political symbols, avoid religious symbols, avoid human remains, avoid weapons, avoid gore. Make it visually fresh and publicly displayable.
