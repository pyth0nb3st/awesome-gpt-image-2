Full Prompt

Create one polished public-gallery concept image titled exactly "Derived Play: Pattern Proof Bingo".

Use case: derived play mechanic / garment production QA game / sewing-pattern review.
Asset type: landscape 16:9 printable paper-pattern challenge sheet for a GPT Image 2 gallery.

Primary request: Derive a playable inspection game from Tech Pack Translator. Players inspect a fictional overshirt sewing pattern and mark assembly or print-readiness defects before the pattern can be printed. This must be a no-people, no-brand paper-pattern QA game, not a tabletop board game, not a fashion ad, not a marketplace listing, not a dashboard, not a menu, not a chart, not a map, and not a 2x3 contact sheet.

Composition: a large central spread of invented overshirt pattern pieces in black technical line art: front left, front right, back, sleeve, yoke, collar, collar stand, cuff, placket, pocket, pocket bag, and flap. Show blue grainline arrows, dashed seam allowance lines, black notches, match points, and a 5 cm scale square. Around the spread, include five inspection zones with short labels: "MISSING NOTCH", "MIRRORED POCKET", "SEAM ALLOWANCE", "GRAINLINE", "SCALE SQUARE". Some defects are circled in red pencil. Include a green "PASS" stamp only on one corrected piece. Add a right-side repair note rail and blank checkbox squares.

Visual style: practical print-proof game sheet, off-white paper, crisp black pattern lines, blue arrows, red pencil correction circles, green pass stamp, subtle scanner grain, clean margins, short legible text. The derived play should be obvious: GPT Image pattern outputs become a human inspection game for assembly logic and print readiness.

Safety and originality constraints: fictional garment only; no real apparel brand, no logo, no marketplace listing, no celebrity/model/person, no face, no mannequin, no protected costume, no cosplay, no real designer name, no real body measurements, no medical/legal/safety claim, no public figures, no politics/elections/parties, no harmful instructions, no adult/explicit content, no gore, no copyrighted characters, no living-artist style imitation. Do not copy any external prompt, sewing pattern, dataset sample, article image, product UI, source image, artwork, garment design, layout, or protected style.
