Full Prompt

Create one original GPT Image 2 use-case concept image as a 16:9 landscape scene.

Title inside the image, small and legible: "GPT Image 2 Reference Loom".

Core format: a single upright studio weaving loom reimagined as an image-generation planning instrument. It uses colored threads, paper image swatches, mask ribbons, crop heddles, label needles, and proofing cloth to show how text prompts, high-fidelity image inputs, edits, masks, and output customization can be combined. This must be one continuous loom object, not a contact sheet, not a 2x3 grid, not separate cards, not a circular rules machine, not pinball, not a film stage, not a greenhouse, not a weather station, not a tabletop route game, not a transparent-background demo.

Six original use-case stations integrated into the loom as labels on physical parts, not boxed panels:
1. "reference-warp thread" keeps fictional input-image traits stable across new generations.
2. "mask-weft edit" weaves a local edit through only one selected area of a fictional scene.
3. "crop-heddle router" tests wide, square, and vertical output sizes without losing the hero subject.
4. "text-label embroider" checks short fictional labels stitched into signs, tags, and package mockups.
5. "selvedge consistency check" compares a sequence of generated variants for character, object, and palette drift.
6. "compression swatch proof" previews the same image idea at crisp archive quality and smaller web-friendly export quality.

Visual language: sophisticated craft-studio product photography with precise technical annotation; maple loom frame, cotton threads, brass heddles, ivory proof cloth, graphite labels, cyan guide strings, coral edit ribbons, slate-gray tools. All sample products, characters, signage, places, and data are fictional.

Safety and originality constraints: no real brands, no trademarks, no celebrities, no public figures, no copyrighted characters, no living artist style imitation, no political persuasion, no elections, no party symbols, no harmful or illegal instructions, no sexual content, no nudity, no explicit content, no graphic violence, no gore, no blood, no weapons, no watermark.

Composition: low frontal three-quarter view of the loom, with thread paths visibly connecting prompt notes, fictional reference swatches, mask ribbons, size proof cloth, and final woven image sample. The six stations should be discoverable as part of the mechanism, not as six equal panels.
