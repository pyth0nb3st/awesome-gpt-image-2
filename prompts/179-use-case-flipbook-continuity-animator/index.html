Full Prompt

Create a single polished gallery image for a new GPT Image 2 use case titled Use Case Flipbook: Continuity Animator. Show a fan-open spiral flipbook lying on a clean studio table, not a contact sheet and not a research board. Each page is a sequential frame of the same invented clockwork fruit courier character walking across a tiny fictional street, with translucent onion-skin overlays, pose tick marks, crop window cutouts, caption beat strips, palette hold tabs, retake slips, and a final thumb-flip preview strip along the edge. The mechanism is using GPT Image 2 for multi-image continuity planning: prompt clauses become page annotations that preserve silhouette, prop identity, readable captions, palette, and crop rhythm across frames. Readable labels should include Seed Frame, Onion Skin, Pose Tick, Caption Beat, Palette Hold, Crop Gate, Retake Slip, and Final Flip. Output format: premium macro photograph of a handmade animation continuity flipbook with vellum overlays, brass spiral binding, graphite frame numbers, crisp legible labels, no hands, no logos, no real brands, no real people, no copyrighted characters, no political content, no dangerous instructions, no gore, no sexual content, no living artist style imitation.
