Create one polished public-gallery concept image titled exactly "Use Case: Daypart Signage Loop".

Use case: productivity-visual / ads-marketing / digital signage workflow.
Asset type: vertical 9:16 fictional public digital-signage deployment mockup, suitable for a GPT Image 2 use-case gallery.

Primary request: Show GPT Image 2 being used to generate a ready-to-deploy dayparted digital-signage loop for a fictional community arts center. The image must feel like a real operations preview for one lobby display that changes by time of day, not a dashboard, not a map, not a menu, not a contact sheet, not a tabletop object, and not a generic poster board.

Composition: A single tall digital screen dominates the frame in a quiet fictional lobby called "Northstar Commons". The screen is split into three stacked time states with soft separators, like one animated playlist preview frozen in one frame: "08:30 Morning Studio", "14:00 Repair Clinic", "19:30 Quiet Reading". Each state has a different small original abstract illustration, a short readable headline, one icon, and a clear room cue such as "Room A", "Workshop", "Reading Hall". Add a thin deployment rail on the right edge with compact labels: "9:16", "Palette", "Time Slot", "Glance Check", "Publish".

Visual style: refined environmental graphic design, matte display surface, tactile warm paper-like textures inside the screen graphics, accessible contrast, clean sans-serif type, calm but not monochrome palette with teal, coral, lemon, black, and soft white. The lobby background should be simple and fictional, with no real architecture or brand cues. No people.

Show the use-case mechanism clearly: one prompt produces multiple time-aware public-screen messages that keep a coherent fictional identity while changing copy, icon, and illustration by time slot. Include tiny QA chips at the bottom of the screen: "Readable", "On Time", "Icon Clear", "Brand Safe".

Safety and originality constraints: no real brands, no logos, no public figures, no politics/elections/parties, no emergency evacuation instructions, no medical/legal claims, no harmful instructions, no adult or explicit content, no gore, no celebrity likeness, no copyrighted characters, no living-artist style imitation. Do not copy any external prompt or image. Keep all visible text short and legible.
