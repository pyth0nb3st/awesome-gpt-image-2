## Maintainer Notes

Generated image binaries are served from Cloudflare R2 rather than committed to Git. The repository keeps metadata, static-site source, thumbnail dimensions, and deployment workflow files small and reviewable.

`gallery.json` is the canonical metadata source for public examples. `assets/thumbs/manifest.json` stays tracked because the static site imports thumbnail dimensions. Pushes to `main` build the static export and publish `out` to the `gh-pages` branch.
